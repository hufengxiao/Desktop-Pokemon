# 快速开始指南

## Windows 用户（推荐方式）

### 步骤 1: 安装依赖

双击运行 `install.bat` 文件，它会自动安装所需的 Python 包。

### 步骤 2: 启动程序

双击运行 `run.bat` 文件即可启动桌宠！

---

## 所有平台（手动方式）

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python src/main.py
```

---

## 首次运行

程序会显示一个蓝色的圆形占位符小怪兽，它会：

- 循环播放"呼吸"动画（idle）
- 点击时跳起来（click）
- 可以拖拽移动
- 右键退出

## 替换为你的角色

准备两个 GIF 动画：

1. **idle.gif** - 待机循环动画
2. **click.gif** - 点击反馈动画

放入 `assets/sprites/pikachu/` 目录，替换现有文件，重启程序即可。

### 动画要求

- 格式：GIF（支持透明背景）
- 推荐尺寸：128x128 或 256x256 像素
- 背景：必须透明（使用 Alpha 通道）
- idle.gif：设置为循环播放
- click.gif：播放一次后会自动返回 idle

## 常用操作

| 操作 | 说明 |
|------|------|
| 左键拖拽 | 移动小怪兽位置 |
| 左键点击 | 触发互动动画 |
| 右键菜单 | 显示菜单（退出等） |

---

## 故障排查

### 问题：程序无法启动

检查是否安装了 Python 3.10+：
```bash
python --version
```

### 问题：看不到窗口

删除 `config.json` 文件并重启程序。

### 问题：动画不显示

确保 `assets/sprites/pikachu/idle.gif` 文件存在且未损坏。
